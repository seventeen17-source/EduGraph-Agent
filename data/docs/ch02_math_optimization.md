## 2.1 线性代数基础

```text
knowledge_node_id: ml_linear_algebra_basic
source_ids: book_mml_deisenroth, book_d2l
title: 线性代数基础的核心作用
importance: 0.80
content: 【核心概念】线性代数为机器学习提供描述数据和模型参数的核心语言。一个样本可以表示为一个向量（一维数组），一批样本组成矩阵（每行是一个样本），模型权重写成向量 w 或矩阵 W，矩阵乘法表达了"将输入向量通过线性变换映射到输出"的运算。【核心运算】向量点积 w·x = Σw_ix_i 用于计算相似度或加权求和；矩阵乘法 Y = XW 表示同时对多个样本做线性变换；范数 ||x|| 表示向量的"长度"或"大小"，L2 范数（欧氏距离）和 L1 范数（曼哈顿距离）是最常用的两种；矩阵的逆 (X^TX)^{-1} 在线性和岭回归的闭式解中出现，要求 X^TX 可逆（满秩）。【直觉理解】如果把每个样本看成空间中的一个点，特征就是坐标轴，向量记录位置，矩阵则像一个变换工具，可以旋转、拉伸或重新组合这些点——线性变换的本质就是对空间做可还原的形变。【与前置知识的关系】这是数学入口知识，不要求完整证明，但需要熟练掌握向量、矩阵、维度和基本运算，才能读懂线性回归（Xw=y 的矩阵形式）、神经网络前向传播（Y = σ(XW + b)）和批量数据计算。【典型误区】矩阵乘法不满足交换律：AB ≠ BA，要特别注意乘法的左右顺序。向量点积和元素积（哈达玛积）是不同的运算。矩阵形状（维度）必须匹配才能相乘——形状不兼容是最常见的线性代数报错原因。【应用场景】线性模型预测、神经网络层计算、降维（PCA 的特征值分解）和相似度度量（余弦相似度 = (a·b)/(|a||b|)）都依赖线性代数表示。
keywords: 向量, 矩阵, 范数, 矩阵乘法, 点积, 线性变换, 维度, L2范数, L1范数
```

## 2.2 概率论与统计基础

```text
knowledge_node_id: ml_probability_statistics_basic
source_ids: book_mml_deisenroth, book_prml_bishop
title: 概率统计在机器学习中的作用
importance: 0.80
content: 【核心概念】概率论与统计用于描述不确定性以及从有限数据中估计规律。随机变量 X 表示不确定的结果，离散随机变量用概率质量函数（PMF）P(X=x) 描述取值概率，连续随机变量用概率密度函数（PDF）p(x)，其积分（而非函数值本身）表示落在区间的概率。条件概率 P(A|B) 描述已知 B 发生时 A 发生的概率；贝叶斯定理 P(A|B) = P(B|A)P(A)/P(B) 用于从结果反向推断原因。期望 E[X] = Σx·P(X=x)（离散）或 ∫x·p(x)dx（连续）是随机变量的平均值；方差 Var(X) = E[(X − E[X])²] 刻画波动程度。【直觉理解】机器学习面对的是有限样本，模型看到的只是总体规律的一个片段。概率统计帮助我们区分"偶然波动"和"稳定规律"——大量样本的平均值趋向于期望，统计学就是研究这种大数定律背后的规律。【与前置知识的关系】先理解样本、特征和标签，再理解概率分布，才能继续学习最大似然估计（MLE）、朴素贝叶斯分类、逻辑回归的交叉熵损失和泛化误差的概率估计。【典型误区】把训练集上的频率直接当成永远不变的真实概率——小样本时频率波动很大。概率密度函数的值本身不是概率，必须积分才有概率意义。独立和不相关是两个不同的概念：独立 ⇒ 不相关，但反过来不一定成立。【应用场景】分类概率估计（逻辑回归输出概率）、朴素贝叶斯分类器、交叉熵损失函数（衡量两个概率分布的差异）和模型评估（置信区间、显著性检验）都需要概率统计基础。
keywords: 概率, 随机变量, 条件概率, 贝叶斯定理, 期望, 方差, 最大似然估计, PMF, PDF
```

## 2.3 微积分与优化基础

```text
knowledge_node_id: ml_calculus_optimization_basic
source_ids: book_mml_deisenroth, book_d2l, book_pumpkin_ml
title: 微积分与优化基础的直觉
importance: 0.90
content: 【核心概念】微积分关心函数如何随输入变化而变化。一元函数中，导数 f'(x) 描述 f 在 x 处的瞬时变化率（斜率）；多参数函数用偏导数 ∂f/∂x_i 描述单独一个方向的变化；梯度 ∇f = (∂f/∂x_1, ∂f/∂x_2, …, ∂f/∂x_n) 是一个向量，指向函数上升最快的方向。优化则是在给定目标函数后，寻找让目标尽可能小（或大）的参数配置。【关键工具】链式法则是求复合函数导数的核心工具：若 y = f(g(x))，则 dy/dx = f'(g(x)) · g'(x)；多层神经网络和反向传播都建立在链式法则之上。Hessian 矩阵（二阶偏导矩阵）描述曲率，用于判断临界点是局部最小、局部最大还是鞍点。【直觉理解】训练模型像调节一组旋钮（参数），损失函数告诉你当前设置下模型表现有多差，梯度告诉你每个旋钮往哪个方向拧会让损失变化最快——往梯度反方向拧（步长乘以负梯度）就能让损失降低。【与前置知识的关系】需要先理解函数、参数和模型输出的含义，再学习损失函数（具体的目标函数）；它又是梯度下降和反向传播的直接前置——两者都依赖链式法则和梯度计算。【典型误区】梯度方向是上升方向，不是下降方向——要下山得往梯度反方向走。导数/梯度为 0 不一定意味着找到理想极值点，也可能是局部极小、局部极大或鞍点。【应用场景】线性回归、逻辑回归、神经网络训练和几乎所有可微模型优化都离不开微积分与优化基础。
keywords: 导数, 偏导数, 梯度, Hessian, 链式法则, 最优化, 局部极值, 鞍点, 目标函数
```

## 2.4 损失函数

```text
knowledge_node_id: ml_loss_function
source_ids: book_zhou_ml, book_pumpkin_ml, book_ml_practice_video_2024
title: 损失函数的作用与选择
importance: 0.90
content: 【核心概念】损失函数（Loss Function）把模型预测 ŷ 与真实标签 y 之间的差距转化为一个可优化的数值目标。训练模型时，优化器不断调整参数使损失在训练数据上降低——损失越小，预测越接近真实标签。不同任务需要不同的损失函数：回归任务常用均方误差（MSE）：L = (1/N)∑(ŷ_i − y_i)²；分类任务常用交叉熵损失（Cross-Entropy）：二分类 L = −[y·log(ŷ) + (1−y)·log(1−ŷ)]，多分类 L = −Σy_i·log(ŷ_i)。【直觉理解】损失函数就像教练给运动员打的”错误分”：分数越大说明动作越离谱，运动员（优化器）需要根据这个分数调整动作（参数）。不同运动有不同的评分规则——跑得快比跑得准重要还是跑得准比跑得快重要，这由损失函数决定。【与前置知识的关系】需要先知道样本、标签、预测值和参数的含义；损失函数是梯度下降和反向传播的载体——梯度通常是对损失函数计算的。【典型误区】损失函数不等同于最终评价指标：训练时的损失和最终汇报的指标（如 AUC、F1）可能不完全一致。例如模型可能优化 log 损失但最终用 AUC 评估，两者的最优点不一定相同。不能把一个任务中的损失随意套到另一个任务——MSE 对异常值非常敏感（平方放大），Huber 损失对异常值更鲁棒。【应用场景】线性回归（MSE）、逻辑回归（交叉熵）、支持向量机（Hinge Loss）、神经网络（交叉熵/MSE）训练都必须先明确损失函数。
keywords: 损失函数, 均方误差, 交叉熵, Hinge损失, Huber损失, 分类, 回归, 模型训练
```

## 2.5 梯度与最优化

```text
knowledge_node_id: ml_gradient_optimization_basic
source_ids: book_mml_deisenroth, book_pumpkin_ml, book_d2l
title: 梯度与最优化的训练直觉
importance: 0.90
content: 【核心概念】梯度是多参数函数在当前位置变化最快方向的向量。给定损失函数 J(θ)，梯度 ∇_θ J 指向函数上升最快的方向——这意味着参数沿负梯度方向移动时，损失函数下降最快，这正是梯度下降法的核心原理。在深度学习中，优化器负责根据梯度更新参数，常见选择包括 SGD（原始梯度下降）、带动量的 SGD（引入动量项以加速收敛和平稳更新）、AdaGrad（自适应调整每个参数的学习率）和 Adam（结合动量和自适应学习率，目前最常用）。【直觉理解】如果损失函数像一片起伏不平的山地，梯度指向上升最快的方向；为了让损失下山，需要朝相反方向（负梯度）移动。”一小步”的大小由学习率控制——步子太大可能越过山谷，步子太小走得很慢。动量项相当于给参数增加惯性，让更新方向更平滑。【与前置知识的关系】必须先理解损失函数（要优化的目标）和微积分中的梯度概念，才能进入梯度下降和反向传播。反向传播负责高效计算梯度（∇_θ J），优化器负责用梯度更新参数（θ ← θ − α·∇_θ J）。【典型误区】梯度方向是上升方向，不是下降方向；局部梯度下降也不保证到达全局最优，学习率过大会震荡。SGD 的”随机”在于用单个或少量样本近似真实梯度，而非随机乱走。Adam 虽然最常用，但在某些收敛理论上不如 SGD，不一定是所有任务的最优选择。【关键公式】梯度下降：θ_{t+1} = θ_t − α∇_θ J(θ_t)；带动量的更新：v_t = βv_{t-1} + (1−β)∇_θ J(θ_t)，θ_{t+1} = θ_t − αv_t。【应用场景】梯度优化思想贯穿线性模型、深度神经网络和大多数现代机器学习训练算法。
keywords: 梯度, 最优化, 参数更新, 迭代, 学习率, 梯度下降, SGD, Adam, 动量
```
