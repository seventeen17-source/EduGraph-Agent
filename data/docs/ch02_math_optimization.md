# 第 2 章 数学基础与优化基础

## 章节定位

本章不完全对应《机器学习》的独立章节，但它是 GraphRAG 学习路径规划的重要前置层，用于支撑线性模型、梯度下降和反向传播。

## 优先级

P0。正式收集时优先补充，但内容应保持够用即可，不展开成完整数学课程。

## 2.1 线性代数基础

```text
knowledge_node_id: ml_linear_algebra_basic
source_ids: book_mml_deisenroth, book_d2l
title: 线性代数基础的核心作用
importance: 0.80
content: 【核心概念】线性代数为机器学习提供描述数据和模型参数的语言：一个样本可以表示为向量，一批样本可以组成矩阵，模型权重也常写成向量或矩阵。矩阵乘法、转置、范数和线性变换用于表达特征组合、模型预测和误差大小。【直觉理解】如果把每个样本看成空间中的一个点，特征就是坐标轴，向量记录位置，矩阵则像一个变换工具，可以旋转、拉伸或重新组合这些点。【与前置知识的关系】这是数学入口知识，不要求完整证明，但需要先理解向量、矩阵和维度，才能读懂线性回归、神经网络前向传播和批量数据计算。【典型误区】不要把矩阵当成普通数字表格；矩阵形状、乘法顺序和维度匹配会直接影响计算是否成立。【应用场景】线性模型的预测、神经网络层计算、降维方法和相似度度量都依赖线性代数表示。
keywords: 向量, 矩阵, 范数, 线性变换, 维度, 矩阵乘法
```

## 2.2 概率论与统计基础

```text
knowledge_node_id: ml_probability_statistics_basic
source_ids: book_mml_deisenroth, book_prml_bishop
title: 概率统计在机器学习中的作用
importance: 0.80
content: 【核心概念】概率论与统计用于描述不确定性和从数据中估计规律。随机变量表示不确定结果，概率分布描述可能取值及其可能性，期望和方差刻画平均趋势与波动程度，条件概率描述已知某些信息后事件发生概率的变化。【直觉理解】机器学习面对的是有限样本，模型看到的只是总体规律的一部分。概率统计帮助我们区分偶然波动和稳定规律，也帮助模型给出带不确定性的判断。【与前置知识的关系】先理解样本、特征和标签，再理解概率分布，才能继续学习最大似然、贝叶斯分类器、交叉熵损失和泛化误差估计。【典型误区】不要把训练集上的频率直接当成永远不变的真实概率；样本量太小时，统计估计可能很不稳定。【应用场景】分类概率估计、朴素贝叶斯、逻辑回归、模型评估和概率图模型都需要概率统计基础。
keywords: 概率, 随机变量, 条件概率, 期望, 方差, 最大似然
```

## 2.3 微积分与优化基础

```text
knowledge_node_id: ml_calculus_optimization_basic
source_ids: book_mml_deisenroth, book_d2l, book_pumpkin_ml
title: 微积分与优化基础的直觉
importance: 0.90
content: 【核心概念】微积分关心函数如何随输入变化而变化，导数描述一元函数的局部变化率，偏导数和梯度把这个思想推广到多参数函数。优化则是在给定目标函数后，寻找让目标尽可能小或尽可能大的参数。【直觉理解】训练模型像调节一组旋钮，损失函数告诉你当前表现有多差，梯度告诉你每个旋钮往哪个方向调会让损失变化最快。【与前置知识的关系】需要先理解函数、参数和模型输出，再学习损失函数；它又是梯度下降和反向传播的直接前置，因为二者都依赖链式法则和梯度计算。【典型误区】梯度不是参数更新本身，而是局部变化方向的信息；导数为零也不一定意味着找到理想最优点，可能是局部最优或鞍点。【应用场景】线性回归、逻辑回归、神经网络训练和几乎所有可微模型优化都离不开微积分与优化。
keywords: 导数, 偏导数, 梯度, 最优化, 链式法则, 目标函数
```

## 2.4 损失函数

```text
knowledge_node_id: ml_loss_function
source_ids: book_zhou_ml, book_pumpkin_ml, book_ml_practice_video_2024
title: 损失函数的作用与选择
importance: 0.90
content: 【核心概念】损失函数把模型预测与真实标签之间的差距转化为一个可优化的数值目标。训练模型时，算法通常不是直接追求“看起来更聪明”，而是不断调整参数，让损失函数在训练数据上降低。【直觉理解】可以把损失看成模型犯错的分数，分数越大说明预测越偏离真实结果；不同任务需要不同计分规则，回归常用均方误差，分类常用交叉熵或对数损失。【与前置知识的关系】需要先知道样本、标签、预测值和参数的含义；它又连接到梯度与最优化，因为梯度通常是围绕损失函数计算的。【典型误区】损失函数不等同于最终评价指标，训练优化的目标和汇报给用户的指标可能不同；也不能把一个任务中的损失随意套到另一个任务。【应用场景】线性回归、逻辑回归、支持向量机和神经网络训练都必须先明确损失函数。
keywords: 损失函数, 目标函数, 均方误差, 交叉熵, 误差, 模型训练
```

## 2.5 梯度与最优化

```text
knowledge_node_id: ml_gradient_optimization_basic
source_ids: book_mml_deisenroth, book_pumpkin_ml, book_d2l
title: 梯度与最优化的训练直觉
importance: 0.90
content: 【核心概念】梯度是多参数函数在当前位置变化最快方向的向量。最优化的目标是在参数空间中寻找让损失较小的参数，机器学习训练常通过迭代方式完成：计算当前参数下的梯度，再根据梯度信息更新参数。【直觉理解】如果损失函数像一片地形，梯度指向上升最快的方向；为了让损失下降，通常要朝相反方向移动一小步。这个“一小步”的大小由学习率等策略控制。【与前置知识的关系】必须先理解损失函数和微积分中的梯度概念，才能进入梯度下降、SGD 和反向传播。反向传播负责高效算梯度，优化算法负责用梯度更新参数。【典型误区】梯度方向本身是上升方向，不是下降方向；局部下降也不保证一步到达全局最优，学习率过大还可能震荡。【应用场景】梯度思想贯穿线性模型、深度神经网络和大多数现代机器学习训练算法。
keywords: 梯度, 最优化, 参数更新, 迭代, 学习率, 梯度下降
```
