## 3.1 线性回归

```text
knowledge_node_id: ml_linear_regression
source_ids: book_zhou_ml, book_pumpkin_ml, book_hands_on_ml_3e_zh
title: 线性回归的建模思想与训练目标
importance: 0.90
content: 【核心概念】线性回归用特征的线性组合预测连续数值目标，模型通常写成“权重乘特征再加偏置”的形式。训练时需要选择参数，使预测值与真实值之间的差距尽可能小，常见目标是最小化均方误差。【直觉理解】如果把每个特征看成影响结果的因素，线性回归就是给每个因素分配一个影响权重，再把它们加起来得到预测。【与前置知识的关系】需要先理解向量、矩阵和损失函数；它又是学习梯度下降、逻辑回归和神经网络线性层的入口。【典型误区】线性回归不等于只能处理一维直线，多特征时是高维线性关系；训练误差低也不代表泛化一定好。【应用场景】房价预测、销量估计、趋势分析和许多模型的基线方案都常从线性回归开始。
keywords: 线性模型, 回归, 均方误差, 参数估计, 权重, 偏置
```

## 3.2 梯度下降

```text
knowledge_node_id: ml_gradient_descent
source_ids: book_mml_deisenroth, book_pumpkin_ml, book_hands_on_ml_3e_zh
title: 梯度下降的核心思想与学习率影响
importance: 0.95
content: 【核心概念】梯度下降是一种迭代优化方法：先计算当前参数处损失函数的梯度，再沿负梯度方向更新参数，使损失逐步降低。学习率控制每次移动的步长，是影响收敛速度和稳定性的关键超参数。【直觉理解】损失函数可以想成一片地形，梯度指向上坡最快方向；想下山就要朝反方向走，但步子太大可能越过低谷，步子太小又会走得很慢。【与前置知识的关系】需要先理解损失函数、梯度和线性回归中的参数更新；它又是 SGD、神经网络训练和反向传播之后参数更新的基础。【典型误区】学习率不是越大越好；梯度下降在非凸问题中也不保证找到全局最优。【应用场景】线性回归、逻辑回归、神经网络和许多可微模型训练都依赖梯度下降或其变体。
keywords: 梯度, 学习率, 损失函数, 参数更新, 收敛, 负梯度
```

## 3.3 随机梯度下降与 Mini-batch

```text
knowledge_node_id: ml_sgd_minibatch
source_ids: book_d2l, book_hands_on_ml_3e_zh, docs_sklearn_user_guide
title: SGD 与 Mini-batch 的区别
importance: 0.85
content: 【核心概念】批量梯度下降每次用全部训练样本计算梯度，SGD 每次用单个样本估计梯度，Mini-batch 则用一小批样本估计梯度。Mini-batch 在计算效率和梯度稳定性之间折中，是现代深度学习训练的常用方式。【直觉理解】全量计算像每次听完整个班的意见再行动，稳定但慢；单样本 SGD 像只听一个人的意见，快但噪声大；Mini-batch 像听一小组意见，通常更均衡。【与前置知识的关系】需要先理解梯度下降和损失函数，才能明白为什么不同 batch size 会影响更新方向和训练速度。【典型误区】SGD 的“随机”不是随便更新，而是用随机抽取的样本近似整体梯度；batch 越大也不一定越好，可能增加计算开销并影响泛化。【应用场景】大规模线性模型、神经网络和深度学习框架训练几乎都使用 Mini-batch。
keywords: SGD, Mini-batch, 批量大小, 随机优化, 梯度估计, 深度学习训练
```

## 3.4 逻辑回归

```text
knowledge_node_id: ml_logistic_regression
source_ids: book_zhou_ml, book_lihang_statistical_learning, book_pumpkin_ml, book_hands_on_ml_3e_zh
title: 逻辑回归为什么用于分类
importance: 0.85
content: 【核心概念】逻辑回归虽然名字里有“回归”，但常用于分类。它先计算线性模型输出，再通过 Sigmoid 或 Softmax 将结果映射为类别概率，训练时常使用对数损失或交叉熵来让正确类别概率更高。【直觉理解】线性回归直接预测数值，而逻辑回归像在数值输出后加了一个概率转换器，把分数压到 0 到 1 之间，再根据阈值或最大概率决定类别。【与前置知识的关系】需要先理解线性回归、概率和损失函数；它也是学习神经网络分类层和交叉熵损失的重要前置。【典型误区】逻辑回归不是回归任务模型；它也不要求原始数据完全线性可分，只是决策边界在特征空间中通常是线性的。【应用场景】二分类、多分类、点击率预测、疾病风险预测和可解释分类基线模型中都常使用逻辑回归。
keywords: 逻辑回归, 分类, Sigmoid, Softmax, 交叉熵, 概率
```

## 3.5 线性判别分析

```text
knowledge_node_id: ml_linear_discriminant_analysis
source_ids: book_zhou_ml, book_lihang_statistical_learning, book_pumpkin_ml
title: 线性判别分析的投影思想
importance: 0.70
content: 【核心概念】线性判别分析（LDA）是一种经典分类与降维方法，核心是寻找一个投影方向，使不同类别投影后的均值尽量分开，同时同一类别内部的样本尽量集中。它关注的是“投影后更容易区分”，而不只是拟合一个预测函数。【直觉理解】如果二维平面上的两类点混在一起，可以换一个观察角度；好的投影方向会让两类点在一条线上分得更开。【与前置知识的关系】需要理解向量、矩阵、均值、方差和分类任务；它和线性回归、逻辑回归同属线性方法，但优化目标不同。【典型误区】LDA 不是普通线性回归，也不只是画一条分类线；它通常依赖类别统计量，对数据分布和类内协方差有一定假设。【应用场景】小样本分类、特征降维、可视化和模式识别中常用 LDA 作为简单且可解释的线性方法。
keywords: 线性判别, LDA, 投影, 类间距离, 类内距离, 降维
```
