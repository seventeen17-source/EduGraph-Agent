## 3.1 线性回归

```text
knowledge_node_id: ml_linear_regression
source_ids: book_zhou_ml, book_pumpkin_ml, book_hands_on_ml_3e_zh
title: 线性回归的建模思想与训练目标
importance: 0.90
content: 【核心概念】线性回归（Linear Regression）用特征的线性组合预测连续数值目标，模型形式为 ŷ = w_1x_1 + w_2x_2 + … + w_nx_n + b，向量形式 ŷ = w^T x + b。训练目标是选择参数 w 和 b，使预测值 ŷ 与真实值 y 之间的差距尽可能小，常见目标是最小化均方误差（Mean Squared Error，MSE）：MSE = (1/N)∑(ŷ_i − y_i)²。求解线性回归有两种常用方法：正规方程（闭式解）直接求 (X^TX)^{-1}X^Ty，以及梯度下降迭代求解。【直觉理解】如果把每个特征看成影响结果的因素，线性回归就是给每个因素分配一个影响权重（正数表示正向影响，负数表示负向影响），再把它们加起来得到预测值。权重的大小反映因素的相对重要性，截距 b 是所有特征都为 0 时的基准值。【与前置知识的关系】需要先理解向量、矩阵（线性代数基础）和损失函数（均方误差）；它又是学习梯度下降（参数如何迭代更新）、逻辑回归（线性模型+激活=分类）和神经网络线性层（深度模型的基本组成）的入口。【典型误区】线性回归不等于只能处理一维直线——多特征时是高维线性关系，在二维空间中表现为直线，在三维中是平面，更多维时是超平面。训练误差低不代表泛化一定好——过拟合同样可能在线性模型中发生，尤其是特征维度接近或超过样本量时。【关键公式】正规方程：w = (X^TX)^{-1}X^Ty；均方误差：MSE = (1/N)∑(y_i − ŷ_i)²。【应用场景】房价预测、销量估计、趋势分析、因果推断中的基线模型和控制变量。sklearn.linear_model.LinearRegression 是最常用的实现。
keywords: 线性回归, 均方误差, 正规方程, 参数估计, 权重, 偏置, MSE, 超平面
```

## 3.2 梯度下降

```text
knowledge_node_id: ml_gradient_descent
source_ids: book_mml_deisenroth, book_pumpkin_ml, book_hands_on_ml_3e_zh
title: 梯度下降的核心思想与学习率影响
importance: 0.95
content: 【核心概念】梯度下降是一种迭代优化方法：先计算当前参数处损失函数的梯度，再沿负梯度方向更新参数，使损失逐步降低。形式化地，若参数向量为 θ，损失为 J(θ)，则更新规则为 θ_{t+1} = θ_t − α ∇J(θ_t)，其中 α > 0 是学习率（步长），∇J 是梯度。学习率 α 控制每次移动的步长，是影响收敛速度和稳定性的关键超参数——过大则震荡甚至发散，过小则收敛极慢。梯度下降可分为批量梯度下降（每次用全部样本计算精确梯度）、随机梯度下降（每次用一个样本近似梯度）和 Mini-batch（用一小批样本折中）。【直觉理解】损失函数可以想成一片地形，梯度指向上坡最快方向；想下山就要朝反方向走，但步子太大可能越过低谷，步子太小又会走得很慢。在凸函数地形中，梯度下降一定可以到达全局最低点；在非凸地形中（如神经网络损失面），通常只能收敛到某个局部最低或鞍点。【与前置知识的关系】需要先理解损失函数（衡量模型预测与真实值差距的函数）、梯度（多元函数的偏导数向量，指向函数上升最快的方向）和线性回归中参数更新的具体形式；它又是 SGD、神经网络训练和反向传播之后参数更新的基础。【典型误区】学习率不是越大越好——过大的学习率会导致参数在极小点附近来回跳动甚至发散；梯度下降在非凸问题中也不保证找到全局最优。另一个常见误解是认为梯度为 0 就一定是极小点，其实还可能是鞍点。【关键公式】参数更新：θ_{new} = θ_{old} − α · ∇_θ J(θ)。【应用场景】线性回归、逻辑回归、神经网络和许多可微模型训练都依赖梯度下降或其变体。sklearn 的 SGDClassifier / SGDRegressor、PyTorch 和 TensorFlow 中的 optimizer 都基于梯度下降或其自适应变体（如 Adam）。
keywords: 梯度, 学习率, 损失函数, 参数更新, 收敛, 负梯度, 批量GD, SGD, Mini-batch, Adam
```

## 3.3 随机梯度下降与 Mini-batch

```text
knowledge_node_id: ml_sgd_minibatch
source_ids: book_d2l, book_hands_on_ml_3e_zh, docs_sklearn_user_guide
title: SGD 与 Mini-batch 的区别
importance: 0.85
content: 【核心概念】批量梯度下降每次用全部训练样本计算梯度，SGD 每次用单个样本估计梯度，Mini-batch 则用一小批样本估计梯度。Mini-batch 在计算效率和梯度稳定性之间折中，是现代深度学习训练的常用方式。【直觉理解】全量计算像每次听完整个班的意见再行动，稳定但慢；单样本 SGD 像只听一个人的意见，快但噪声大；Mini-batch 像听一小组意见，通常更均衡。【与前置知识的关系】需要先理解梯度下降和损失函数，才能明白为什么不同 batch size 会影响更新方向和训练速度。【典型误区】SGD 的“随机”不是随便更新，而是用随机抽取的样本近似整体梯度；batch 越大也不一定越好，可能增加计算开销并影响泛化。【应用场景】大规模线性模型、神经网络和深度学习框架训练几乎都使用 Mini-batch。
keywords: SGD, Mini-batch, 批量大小, 随机优化, 梯度估计, 深度学习训练
```

## 3.4 逻辑回归

```text
knowledge_node_id: ml_logistic_regression
source_ids: book_zhou_ml, book_lihang_statistical_learning, book_pumpkin_ml, book_hands_on_ml_3e_zh
title: 逻辑回归为什么用于分类
importance: 0.85
content: 【核心概念】逻辑回归虽然名字里有”回归”，但常用于分类。它先计算线性模型输出 z = w·x + b，再通过 Sigmoid 函数 σ(z) = 1/(1+e^{−z}) 将实数 z 映射为 0 到 1 之间的概率值，表示属于正类的置信度。多分类时常用 Softmax 函数代替 Sigmoid，将多个 logit 归一化为概率分布。训练时使用对数损失（binary cross-entropy）或交叉熵来让正确类别概率更高。模型预测时，取概率大于阈值（常为 0.5）的类别作为输出，或取概率最大的类别。【直觉理解】线性回归直接预测数值，而逻辑回归像在数值输出后加了一个概率转换器，把分数压到 0 到 1 之间，再根据阈值或最大概率决定类别。Sigmoid 的 S 形曲线将很大的正数压向 1、很大的负数压向 0，在中间区域变化最敏感，适合做概率估计。【与前置知识的关系】需要先理解线性回归（知道线性组合 w·x+b 是什么）、概率和损失函数（理解什么在驱动训练）；它也是学习神经网络分类层和交叉熵损失的重要前置。理解逻辑回归的梯度推导（对数损失对 w 的偏导）是理解神经网络反向传播的极好入口。【典型误区】逻辑回归不是回归任务模型，它输出的”回归系数”实际上是分类决策边界；它也不要求原始数据完全线性可分，只是决策边界在特征空间中通常是线性的——如果数据本身非线性，可以先做特征工程（多项式特征、核方法）再套逻辑回归。【关键公式】Sigmoid：σ(z) = 1/(1+e^{−z})；对数损失：L = −[y·log(σ(z)) + (1−y)·log(1−σ(z))]。【应用场景】二分类（疾病诊断、违约预测、垃圾邮件检测）、多分类、点击率预测和作为可解释分类基线模型。sklearn.linear_model.LogisticRegression 和 statsmodels 是常用实现。
keywords: 逻辑回归, 分类, Sigmoid, Softmax, 交叉熵, 对数损失, 概率, 决策边界
```

## 3.5 线性判别分析

```text
knowledge_node_id: ml_linear_discriminant_analysis
source_ids: book_zhou_ml, book_lihang_statistical_learning, book_pumpkin_ml
title: 线性判别分析的投影思想
importance: 0.70
content: 【核心概念】线性判别分析（Linear Discriminant Analysis，LDA）由 Ronald Fisher 提出，是一种经典的有监督分类与降维方法。LDA 的核心是寻找一个投影方向 w，使得不同类别投影后的类间均值差异最大化，同时同一类别内部的样本方差最小化。对于二分类问题，目标为最大化 J(w) = (|μ_1 − μ_2|)² / (σ_1² + σ_2²)，其中 μ_i 是第 i 类的投影均值，σ_i² 是投影方差。与逻辑回归不同，LDA 直接建模每个类别的条件概率密度，假设各类别服从同方差的高斯分布，然后基于贝叶斯决策进行分类。【直觉理解】如果二维平面上的两类点混在一起，可以换一个观察角度（投影方向）——好的投影方向会让两类点在一条线上的分布分得更开，每类内部尽可能紧凑。Fisher 的核心洞见是把”分类”问题转化为”投影后类间差异与类内差异之比最大化”的问题。【与前置知识的关系】需要理解向量、矩阵、均值、方差（统计学基础）和分类任务；它和线性回归、逻辑回归同属线性方法，但优化目标和假设不同——LDA 是生成式方法（建模每个类的概率密度），逻辑回归是判别式方法（直接建模后验概率）。【典型误区】LDA 不是普通线性回归的变种，它的优化目标完全不同。LDA 通常依赖各类别协方差矩阵相等的假设，在协方差差异大时表现会下降，此时二次判别分析（QDA）是更好的选择。LDA 也是一种降维方法（降到 K−1 维，K 为类别数），这在可视化场景中很有用。【应用场景】小样本二分类、特征降维（降维后分类）、数据可视化、作为分类的简单可解释模型和作为其他方法的初始化基线。
keywords: 线性判别, LDA, Fisher, 投影, 类间距离, 类内方差, 生成式模型, 降维
```
