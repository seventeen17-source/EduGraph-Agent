## 6.1 感知机

```text
knowledge_node_id: ml_perceptron
source_ids: book_zhou_ml, book_lihang_statistical_learning, book_nndl_qiu
title: 感知机作为神经元模型入口
importance: 0.75
content: 【核心概念】感知机（Perceptron）是 Frank Rosenblatt 于 1957 年提出的早期神经元模型，是神经网络和深度学习的起点。它用一组权重 w_1, w_2, …, w_n 对输入特征 x_1, x_2, …, x_n 加权求和，再与阈值 θ 比较，输出类别判断。形式化为：若 w·x ≥ θ 则输出 +1，否则输出 −1。感知机学习算法通过迭代调整权重来减少分类错误，在训练数据线性可分时可以证明收敛。【直觉理解】感知机像一个简单判定器：每个特征给出支持或反对的证据，权重表示证据强弱，综合分数超过阈值就判为某一类。把多个感知机组合起来就像多人投票，每个人关注不同证据，最后综合决策。【与前置知识的关系】需要先理解线性模型（知道加权求和是什么）、逻辑回归（知道阈值/Sigmoid 的分类作用）和分类任务；它是理解多层神经网络的入口——多个感知机组合可形成隐藏层，组合更多层就能处理非线性问题。【典型误区】感知机不等于神经网络本身，单层感知机只能处理线性可分问题（如 AND、OR 但不是 XOR）；多层感知机（MLP）通过叠加层才能解决 XOR 等非线性问题。感知机的收敛性也依赖于数据线性可分的前提。【关键公式】感知机判别：f(x) = sign(w·x − θ)。【应用场景】感知机本身现已少用，主要作为理解神经网络发展历史和反向传播机制的教学工具，以及线性二分类问题的简单基线。
keywords: 感知机, 神经元, 线性分类, 权重, 阈值, Rosenblatt, XOR问题, MLP
```

## 6.2 多层神经网络

```text
knowledge_node_id: ml_multilayer_neural_network
source_ids: book_d2l, book_nndl_qiu, book_hands_on_ml_3e_zh
title: 多层神经网络的结构与前向传播
importance: 0.85
content: 【核心概念】多层神经网络（Multi-Layer Perceptron，MLP）由输入层、一个或多个隐藏层和输出层组成。每一层将上一层输出作为输入，先做线性变换 z = Wx + b，再通过非线性激活函数 a = σ(z) 得到该层的输出，这个过程称为前向传播（Forward Propagation）。隐藏层数称为"深度"，深度越大模型表达能力理论上越强，但也更难训练。【直觉理解】单层线性模型只能画一条直线作为分类边界，多层网络像逐步加工信息的流水线：前面层提取基础特征（边缘、纹理），后面层组合这些基础特征形成更抽象的表示（物体部件、整体类别）。【与前置知识的关系】需要先理解感知机（知道神经元的基本计算单元）、矩阵乘法（层间变换的数学形式）和激活函数（非线性引入的关键）；它又是反向传播、CNN、RNN 和 Transformer 的共同基础。【典型误区】层数增加不必然带来效果提升——层数过深会导致梯度消失或爆炸，增加训练难度。另一个误区是认为"有了多层就可以自动拟合任何函数"——没有非线性激活函数时，多层线性变换在数学上等价于单层线性变换（因为矩阵乘法可以合并），因此激活函数是深层网络能够表达复杂模式的必要条件。【关键公式】第 l 层前向传播：z^{(l)} = W^{(l)}a^{(l-1)} + b^{(l)}，a^{(l)} = σ(z^{(l)})。【应用场景】MLP 用于分类、回归、表征学习，是几乎所有现代深度学习模型（CNN、RNN、Transformer）的通用底层框架。PyTorch 的 torch.nn.Linear 或 Keras 的 Dense 层都是 MLP 的实现。
keywords: 多层神经网络, 隐藏层, 前向传播, MLP, 非线性, 深度, 矩阵乘法
```

## 6.3 激活函数

```text
knowledge_node_id: ml_activation_function
source_ids: book_d2l, book_nndl_qiu
title: 激活函数为什么引入非线性
importance: 0.80
content: 【核心概念】激活函数（Activation Function）作用在神经元（或一层）的线性输出之后，为网络引入非线性。如果没有激活函数，多层网络只是多个线性变换的叠加，可以合并为单层线性变换，表达能力与单层网络等价。常见激活函数包括：Sigmoid σ(x) = 1/(1+e^{−x}) 将输出压缩到 (0,1)；Tanh tanh(x) = (e^x − e^{−x})/(e^x + e^{−x}) 将输出压缩到 (−1,1)；ReLU f(x) = max(0, x) 是最常用的现代激活函数，计算高效且能缓解梯度消失。【直觉理解】线性层像只会拉伸和旋转空间，激活函数则像给空间加入折弯能力，使网络能够拟合弯曲复杂的决策边界。没有激活函数的神经网络，无论堆多少层，本质上只是一个线性模型。【与前置知识的关系】需要先理解感知机（知道线性加权求和的计算）和矩阵乘法（线性变换的表达）；它是多层神经网络能够表达复杂模式的关键，也直接影响反向传播中梯度的传递方式。【典型误区】激活函数不是随便选的装饰项，不同激活函数的梯度特性差异很大：Sigmoid 在输入绝对值较大时梯度接近 0，导致深层网络训练困难（梯度消失）；ReLU 虽然高效，但"死 ReLU"问题——部分神经元在训练中梯度永远为 0，不再更新——需要用 Leaky ReLU、ELU 等变体缓解。【关键公式】ReLU：f(x) = max(0, x)，导数：f'(x) = 1 if x > 0 else 0。【应用场景】ReLU 及其变体（Leaky ReLU、PReLU）广泛用于现代神经网络的隐藏层；Sigmoid 和 Softmax 常用于输出层做概率估计。
keywords: 激活函数, Sigmoid, ReLU, Tanh, 非线性, 梯度消失, 死ReLU
```

## 6.4 反向传播

```text
knowledge_node_id: ml_backpropagation
source_ids: book_d2l, book_nndl_qiu, course_stanford_cs231n
title: 反向传播与梯度下降的分工
importance: 0.95
content: 【核心概念】反向传播（Backpropagation，简称 BP）是利用链式法则，从损失函数开始把梯度由输出层逐层传回前面的层，从而高效计算每个参数对损失的影响。它解决的是”如何快速算出每个参数该往哪个方向调整”这个问题，而参数如何根据这些梯度来更新则通常由梯度下降、SGD 或 Adam 等优化算法完成，两者配合构成完整的训练过程。链式法则是反向传播的数学基础：若 y = f(g(x))，则 dy/dx = f'(g(x)) · g'(x)；多层网络的复合函数链可以逐层展开计算。【直觉理解】前向传播像把输入一路加工成预测，每一层都在做一次信息转换；反向传播则像沿加工链路反向追溯责任——最终误差由哪些参数造成，每一层应承担多少变化方向。具体地说，反向传播先算输出层的梯度，再将该梯度作为上游信号传回前一层，每层利用已知的上游梯度和本地激活值计算出本地参数的梯度，再继续向前传。【与前置知识的关系】需要先理解多层神经网络（知道什么是全连接层、激活函数和损失函数）、链式法则（能手动求复合函数导数）、梯度下降（知道梯度指什么、为什么要用负梯度更新）和 Mini-batch（现代训练几乎都使用小批量）。它是训练深度网络的核心算法，理解反向传播后学习 CNN、RNN 和 Transformer 的训练过程会更加顺畅。【典型误区】反向传播不等于梯度下降——反向传播只是计算梯度，梯度下降是用这些梯度更新参数；它也不只适用于很深的网络，只要计算图可微就能使用类似思想。另一个常见误解是认为”梯度消失”是因为反向传播传得太远，实际上梯度消失是因为链式法则的连乘效应导致梯度越来越小，尤其在 Sigmoid 激活和深层网络中更明显——ReLU、残差连接和归一化层都是针对这一问题的改进。【关键公式】链式法则：∂L/∂x^{(l)} = (∂z^{(l+1)}/∂x^{(l)})^T · ∂L/∂z^{(l+1)}。其中 ∂L/∂z^{(l+1)} 是上游传来的梯度，∂z^{(l+1)}/∂x^{(l)} 是本地雅可比矩阵。【应用场景】MLP、CNN、RNN、Transformer 等几乎所有神经网络训练都依赖反向传播。PyTorch 的 autograd 机制和 TensorFlow 的tf.GradientTape都是自动微分实现，底层都是反向传播。
keywords: 反向传播, 链式法则, 梯度, 神经网络训练, 参数更新, 自动微分, 梯度消失
```

## 6.5 Dropout

```text
knowledge_node_id: ml_dropout
source_ids: book_d2l, book_nndl_qiu, book_hands_on_ml_3e_zh
title: Dropout 如何缓解过拟合
importance: 0.70
content: 【核心概念】Dropout 是神经网络中的正则化方法，由 Hinton 等人于 2012 年提出。训练时以概率 p（常取 0.5）随机将部分神经元的输出置零（相当于让这些神经元"缺席"），使网络不能过度依赖某些特定神经元的连接，从而降低过拟合风险。每次前向传播时重新随机决定哪些神经元被丢弃，相当于训练了多个不同的"子网络"的集成。推理阶段不再随机丢弃，而是使用完整网络，并将每个神经元的输出乘以保留概率 p（等效缩放）以保持期望值一致。【直觉理解】Dropout 像训练一个团队时随机让部分成员暂时缺席，迫使其他成员也学会承担任务，最终整体协作更稳健。类比于随机让一部分学生组成小组讨论，下次又随机换人——这样没有谁能主导全场，每个人都得真正理解内容才能参与。【与前置知识的关系】需要先理解多层神经网络（知道什么是神经元和层间连接）和过拟合（正则化的直接动机）；它和 L1/L2 正则化、数据增强、早停一样，都是提升泛化能力的工具，且 Dropout 是专为神经网络设计的。【典型误区】Dropout 不是测试时也随意丢弃神经元——测试阶段必须使用完整网络。Dropout 也不能替代所有正则化方法，在数据量充足时可能不如其他正则化手段有效。丢弃率过高（如 p > 0.8）会导致大部分神经元失活，造成欠拟合。【关键参数】丢弃率 p：常用 0.5（隐藏层），输入层通常 p 较小（0.2 左右）。【应用场景】Dropout 主要用于大型全连接层，以及部分卷积网络的正则化，尤其在模型参数量远大于训练样本量时效果明显。
keywords: Dropout, 正则化, 过拟合, 神经网络, 泛化, 集成, 丢弃率
```

## 6.6 BatchNorm

```text
knowledge_node_id: ml_batchnorm
source_ids: book_d2l, book_nndl_qiu
title: BatchNorm 对训练稳定性的作用
importance: 0.70
content: 【核心概念】BatchNorm（Batch Normalization）由 Sergey Ioffe 和 Christian Szegedy 于 2015 年提出，在训练过程中对小批量（mini-batch）数据的中间激活进行归一化，使其均值为 0、方差为 1，再引入可学习的缩放参数 γ 和平移参数 β 让模型自己决定归一化的程度。公式：μ_B = (1/m)∑x_i，σ²_B = (1/m)∑(x_i − μ_B)²，x̂_i = (x_i − μ_B)/√(σ²_B + ε)，y_i = γx̂_i + β。BatchNorm 使网络训练更稳定，允许使用更大的学习率，并能加快收敛速度。它改变了层间数据分布和梯度传播条件，而不只是简单的预处理。【直觉理解】如果每一层接收到的输入数据尺度不断剧烈变化（协变量偏移），后续层就像总在适应一个移动的靶子；BatchNorm 把每一层的中间表示拉回到一个较稳定的范围，让后续层专注于学习真正的模式，而不是不断适应前层的输出变化。【与前置知识的关系】需要先理解 Mini-batch（BatchNorm 的归一化粒度就是 mini-batch）、多层网络（为什么需要逐层稳定）和梯度传播（归一化如何改善梯度流动）；它常与优化算法和激活函数一起影响训练效果。【典型误区】BatchNorm 在训练阶段和推理阶段的计算方式不同——推理时使用全局统计量（移动均值/方差），不是当前 batch 的统计量。它也不能简单等同于正则化，虽然有时会带来一定的正则化效果。在循环神经网络中直接使用标准 BatchNorm 效果不好，需要配合层归一化（LayerNorm）使用。【应用场景】BatchNorm 在 CNN 和深层全连接网络中广泛使用，是现代深度学习训练的标准组件之一。
keywords: BatchNorm, 批归一化, Mini-batch, 训练稳定性, 梯度消失, 协变量偏移
```

## 6.7 CNN

```text
knowledge_node_id: ml_cnn
source_ids: book_d2l, book_nndl_qiu, course_stanford_cs231n
title: CNN 的局部连接与参数共享
importance: 0.80
content: 【核心概念】卷积神经网络（Convolutional Neural Network，CNN）的核心是卷积层，它通过卷积核（滤波器）在输入数据（如图像）的局部区域滑动进行特征提取，并与全连接网络形成对比。CNN 的两个关键特性是局部连接（每个神经元只连接输入的一个局部区域，而非全部）和参数共享（同一个卷积核在整张图上复用权重）。局部连接降低了参数量，参数共享使卷积核学会了某种模式后可以在任意位置检测该模式——这正契合了图像中边缘、纹理等模式可能出现在任意位置的特点。【直觉理解】识别图片中的猫时，边缘、纹理等局部模式可能出现在图像的任意位置；卷积核像一个小放大镜（感受野），在整张图上滑动寻找相似模式，发现了就说"这里有边缘"。【与前置知识的关系】需要先理解多层神经网络（知道权重、激活和反向传播）和反向传播（训练卷积核）；典型 CNN 由卷积层、池化层和全连接层组成：卷积层提取特征，池化层降维压缩，全连接层做最终分类。【典型误区】卷积不是简单压缩图片——卷积核学习的是有意义的特征表示。池化也不是必须越多越好，过度池化会丢失位置信息。CNN 的优势来自结构先验（局部性和平移不变性），而非仅仅因为参数量更少。【关键概念】感受野（Receptive Field）：卷积核能"看到"的输入区域大小；步长（Stride）：卷积核每次移动的像素数；填充（Padding）：在输入边缘补零以控制输出尺寸。【应用场景】图像分类（ResNet、EfficientNet）、目标检测（YOLO、Faster R-CNN）、图像分割（U-Net）、医学图像分析和视频理解都广泛使用 CNN。
keywords: CNN, 卷积, 池化, 局部连接, 参数共享, 感受野, 图像分类, 特征提取
```

## 6.8 RNN

```text
knowledge_node_id: ml_rnn
source_ids: book_d2l, book_nndl_qiu
title: RNN 如何处理序列数据
importance: 0.70
content: 【核心概念】循环神经网络（Recurrent Neural Network，RNN）用于处理序列数据，核心设计是在每个时间步接收当前输入 x_t，并结合上一个时间步的隐藏状态 h_{t-1} 产生新的隐藏状态 h_t 和输出 o_t。隐藏状态 h_t 像对历史信息的压缩记忆，将过去的输入序列编码进一个固定长度的向量，使模型能够根据上下文进行预测。RNN 的参数在所有时间步共享，每个时间步执行相同的变换，形式化：h_t = σ(W_{xh}x_t + W_{hh}h_{t-1} + b_h)，o_t = σ_out(W_{hy}h_t + b_y)。【直觉理解】阅读一句话时，理解当前词往往依赖前面的词——"他踢了___，球飞走了"，要填空格需要前面的"踢"提供信息。RNN 就像一边读一边更新笔记，把过去的信息带到当前的判断中。【与前置知识的关系】需要先理解多层神经网络和反向传播（知道权重、激活和梯度如何计算）；RNN 的训练使用"沿时间展开"后的反向传播（Backpropagation Through Time，BPTT），将循环网络展开成深层前馈网络再计算梯度。【典型误区】RNN 并不能自然记住任意长的历史——在长序列中由于链式法则的连乘效应，容易出现梯度消失（难以记住远期依赖）或梯度爆炸（训练不稳定）。RNN 的隐藏状态也不是完整保存了所有过去输入，而是一种有损压缩。一些常见改进包括 LSTM（门控记忆）、GRU 和注意力机制。【关键公式】RNN 隐藏状态更新：h_t = tanh(W_{xh}x_t + W_{hh}h_{t-1} + b_h)。【应用场景】RNN 曾广泛用于语言建模、时间序列预测、语音识别和早期序列标注任务，但近年来在大多数任务上被 Transformer 及其变体超越。
keywords: RNN, 序列建模, 隐藏状态, 时间步, 梯度消失, BPTT, 语言模型
```

## 6.9 LSTM

```text
knowledge_node_id: ml_lstm
source_ids: book_d2l, book_nndl_qiu
title: LSTM 的门控记忆机制
importance: 0.65
content: 【核心概念】LSTM（Long Short-Term Memory）是 Sepp Hochreiter 和 Jürgen Schmidhuber 于 1997 年提出的 RNN 变体，通过精心设计的门控机制缓解普通 RNN 的梯度消失问题，从而能够学习更长期的依赖关系。LSTM 的核心是记忆单元（Cell State）c_t——一条沿时间轴传递的信息"传送带"，信息可以沿这条传送带流动而几乎不被改变，从而解决远期依赖问题。LSTM 用三个门控制信息流：遗忘门（Forget Gate）f_t 决定保留多少旧记忆；输入门（Input Gate）i_t 决定写入多少新信息；输出门（Output Gate）o_t 决定用记忆单元的多少内容产生当前输出。形式化：f_t = σ(W_f·[h_{t-1}, x_t] + b_f)，i_t = σ(W_i·[h_{t-1}, x_t] + b_i)，C̃_t = tanh(W_C·[h_{t-1}, x_t] + b_C)，C_t = f_t ⊙ C_{t-1} + i_t ⊙ C̃_t，o_t = σ(W_o·[h_{t-1}, x_t] + b_o)，h_t = o_t ⊙ tanh(C_t)。【直觉理解】普通 RNN 像每读一个新词就把笔记完全重写，LSTM 则多了三个开关：哪些旧信息继续保存（遗忘门）、哪些新信息写入（输入门）、当前记忆用于什么（输出门）。【与前置知识的关系】需要先理解 RNN（知道隐藏状态的概念）和梯度消失问题（理解为什么需要门控）；LSTM 是进入注意力机制和 Transformer 序列模型前的重要历史节点。【典型误区】LSTM 不是彻底解决所有长程依赖问题，在极长序列上仍可能表现不足；门控机制增加了参数量和计算成本，在某些简单任务上可能不如更简单的模型。LSTM 的门控含义虽然直观，但实际训练中学到的门控行为往往与直觉不完全一致。【应用场景】文本序列、语音识别、时间序列预测、机器翻译和需要较长上下文记忆的任务曾广泛使用 LSTM。
keywords: LSTM, 门控机制, 长程依赖, 记忆单元, 遗忘门, 输入门, 输出门, 序列模型
```

## 6.10 注意力机制

```text
knowledge_node_id: ml_attention_mechanism
source_ids: book_d2l, book_nndl_qiu, course_hung_yi_lee_ml
title: 注意力机制的加权选择思想
importance: 0.85
content: 【核心概念】注意力机制（Attention Mechanism）由 Dzmitry Bahdanau 等人于 2014 年提出，最初用于改进机器翻译中 RNN 的编码-解码结构。其核心思想是通过为不同输入位置分配权重，让模型在生成当前输出时重点利用更相关的信息，而不必依赖一个固定长度的隐藏状态来压缩全部历史信息。经典的缩放点积注意力（Scaled Dot-Product Attention）定义为：Attention(Q, K, V) = softmax(QK^T/√d_k)V。其中 Q（Query）表示当前要查询的内容，K（Key）表示各位置的索引信息，V（Value）表示各位置的实际内容。Q 和 K 的相似度决定注意力权重，再对 V 做加权求和得到输出。【直觉理解】人在读一段话时，不会平均分配注意力到每个字，而是重点关注与当前问题最相关的句子；注意力机制就是让模型学习这种”按相关性加权读取”的能力。【与前置知识的关系】需要先理解向量点积（计算相似度的数学基础）、矩阵乘法（QK^T 的计算）和序列模型（RNN 的上下文瓶颈是注意力的直接动机）；它是 Transformer 自注意力的直接前置。【典型误区】注意力权重不等于人类可解释性——高注意力权重可能只说明模型”使用了”某位置，不等于该位置”重要”或”正确”。注意力也不天然包含序列位置信息，因为 QK^T 计算的是内容相关性，与位置无关，所以 Transformer 需要额外加入位置编码（Positional Encoding）来表达词序。【关键公式】注意力输出：Attention(Q,K,V) = softmax(QK^T/√d_k)V。【应用场景】注意力机制广泛应用于机器翻译、文本生成、图像描述、语音识别、多模态理解和现代大语言模型（GPT、ChatGPT、Llama 等）都是基于注意力或其变体构建的。
keywords: 注意力机制, Query, Key, Value, 缩放点积注意力, 加权求和, 位置编码, Transformer
```

## 6.11 Transformer

```text
knowledge_node_id: ml_transformer
source_ids: book_d2l, book_nndl_qiu, course_mit_6s191
title: Transformer 的自注意力架构
importance: 0.80
content: 【核心概念】Transformer 是基于自注意力机制的深度学习架构，核心模块包括多头自注意力、前馈网络、残差连接、归一化和位置编码。它不依赖 RNN 的逐步递归结构，而是让序列中不同位置可以直接通过注意力建立联系。【直觉理解】RNN 像按顺序传话，远处信息可能在传递中变弱；Transformer 更像让所有位置开会，每个位置都能根据相关性直接听取其他位置的信息。【与前置知识的关系】需要先理解多层神经网络、注意力机制和位置表示；它是现代大语言模型的重要基础。【典型误区】Transformer 不是不需要训练数据，也不一定在任何小任务上都优于简单模型；自注意力还需要位置编码或类似机制来表达顺序。【应用场景】机器翻译、文本生成、问答系统、代码模型、视觉 Transformer 和多模态大模型都以 Transformer 为核心或重要组件。
keywords: Transformer, 自注意力, 多头注意力, 位置编码, 大模型
```
