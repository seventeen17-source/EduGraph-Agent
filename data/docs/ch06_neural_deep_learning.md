# 第 6 章 神经网络与深度学习基础

## 章节定位

本章对应《机器学习》神经网络章节，并结合《动手学深度学习》和《神经网络与深度学习》补充现代深度学习基础。

## 优先级

P0。正式收集时优先补感知机、多层神经网络、激活函数、反向传播、CNN、Transformer 和注意力机制。

## 6.1 感知机

```text
knowledge_node_id: ml_perceptron
source_ids: book_zhou_ml, book_lihang_statistical_learning, book_nndl_qiu
title: 感知机作为神经元模型入口
importance: 0.75
content: 【核心概念】感知机是早期神经元模型之一，用一组权重对输入特征加权求和，再经过阈值或激活规则输出类别判断。它本质上是线性分类器，只能直接处理线性可分问题。【直觉理解】感知机像一个简单判定器：每个特征给出支持或反对的证据，权重表示证据强弱，综合分数超过阈值就判为某一类。【与前置知识的关系】需要先理解线性模型、逻辑回归和分类任务；它是理解多层神经网络的入口，因为多层网络可以看作许多类似神经元的组合。【典型误区】感知机不是深度网络本身，也不能解决所有非线性分类问题；单个感知机表达能力有限。【应用场景】感知机常用于解释神经网络的基本单元、线性分类和神经网络发展脉络。
keywords: 感知机, 神经元, 线性分类, 权重, 阈值
```

## 6.2 多层神经网络

```text
knowledge_node_id: ml_multilayer_neural_network
source_ids: book_d2l, book_nndl_qiu, book_hands_on_ml_3e_zh
title: 多层神经网络的结构与前向传播
importance: 0.85
content: 【核心概念】多层神经网络由输入层、一个或多个隐藏层和输出层组成。每一层通常先进行线性变换，再通过非线性激活函数得到新的表示；从输入到输出逐层计算的过程称为前向传播。【直觉理解】单层线性模型只能画较简单的边界，多层网络像逐步加工信息的流水线：前面层提取基础特征，后面层组合成更抽象的表示。【与前置知识的关系】需要先理解感知机、矩阵乘法和激活函数；它又是反向传播、CNN、RNN 和 Transformer 的共同基础。【典型误区】层数增加不必然提升效果，模型更复杂也更容易过拟合；没有非线性激活时，多层线性变换仍可合并为一层线性模型。【应用场景】多层神经网络用于分类、回归、表征学习和深度模型的基础模块。
keywords: 多层神经网络, 隐藏层, 非线性, 前向传播, 表征学习
```

## 6.3 激活函数

```text
knowledge_node_id: ml_activation_function
source_ids: book_d2l, book_nndl_qiu
title: 激活函数为什么引入非线性
importance: 0.80
content: 【核心概念】激活函数作用在神经元的线性输出之后，为网络引入非线性。常见激活函数包括 Sigmoid、Tanh 和 ReLU。没有激活函数时，多层网络只是多个线性变换的叠加，表达能力仍接近单层线性模型。【直觉理解】线性层像只会拉伸和旋转空间，激活函数则像给空间加入折弯能力，使网络能拟合弯曲复杂的关系。【与前置知识的关系】需要先理解感知机和线性变换；它是多层神经网络能够表达复杂模式的关键，也影响反向传播中的梯度传递。【典型误区】激活函数不是随便选的装饰项；Sigmoid 在深层网络中可能出现梯度变小，ReLU 虽常用也可能出现部分神经元长期不激活。【应用场景】ReLU 及其变体广泛用于隐藏层，Sigmoid 或 Softmax 常用于输出概率。
keywords: 激活函数, Sigmoid, ReLU, Tanh, 非线性, 梯度
```

## 6.4 反向传播

```text
knowledge_node_id: ml_backpropagation
source_ids: book_d2l, book_nndl_qiu, course_stanford_cs231n
title: 反向传播与梯度下降的分工
importance: 0.95
content: 【核心概念】反向传播利用链式法则，从损失函数开始把梯度由输出层逐层传回前面的层，从而计算每个参数对损失的影响。它解决的是“如何高效算梯度”，而参数如何更新通常由梯度下降、SGD 或 Adam 等优化算法完成。【直觉理解】前向传播像把输入一路加工成预测，反向传播则像沿加工链路追踪责任：最终误差由哪些参数造成，每一层应承担多少变化方向。【与前置知识的关系】需要先理解多层神经网络、损失函数、梯度下降和 Mini-batch；它是训练深度网络的核心算法。【典型误区】反向传播不等于梯度下降；它也不只适用于很深的网络，只要计算图可微就能使用类似思想。【应用场景】MLP、CNN、RNN、Transformer 等神经网络训练都依赖反向传播。
keywords: 反向传播, 链式法则, 梯度, 神经网络训练, 参数更新
```

## 6.5 Dropout

```text
knowledge_node_id: ml_dropout
source_ids: book_d2l, book_nndl_qiu, book_hands_on_ml_3e_zh
title: Dropout 如何缓解过拟合
importance: 0.70
content: 【核心概念】Dropout 是神经网络中的正则化方法，训练时随机将一部分神经元输出置零，使网络不能过度依赖某些特定连接，从而降低过拟合风险。推理阶段通常不再随机丢弃，而是使用完整网络并做相应缩放或采用框架内置处理。【直觉理解】Dropout 像训练一个团队时随机让部分成员暂时缺席，迫使其他成员也学会承担任务，最终整体协作更稳健。【与前置知识的关系】需要先理解多层神经网络和过拟合；它和 L1/L2 正则化、数据增强、早停一样，都是提升泛化能力的工具。【典型误区】Dropout 不是测试时也随意丢神经元；它不能替代所有正则化方法，过高丢弃率还会导致欠拟合。【应用场景】Dropout 常用于全连接层和部分深度网络结构中，尤其在数据较少或模型较大时。
keywords: Dropout, 正则化, 过拟合, 神经网络, 泛化
```

## 6.6 BatchNorm

```text
knowledge_node_id: ml_batchnorm
source_ids: book_d2l, book_nndl_qiu
title: BatchNorm 对训练稳定性的作用
importance: 0.70
content: 【核心概念】BatchNorm 在训练过程中对小批量数据的中间激活进行归一化，并引入可学习的缩放和平移参数，使网络训练更稳定，常能允许更大学习率并加快收敛。它改变的是层间数据分布和梯度传播条件，而不只是简单预处理。【直觉理解】如果每一层接收到的数据尺度不断变化，后续层就像总在适应移动靶；BatchNorm 帮助把中间表示拉回较稳定的范围，再让模型学习合适的缩放。【与前置知识的关系】需要先理解 Mini-batch、多层网络和归一化思想；它常与优化算法和激活函数一起影响训练效果。【典型误区】BatchNorm 不只在推理阶段起作用；它也不能简单等同于正则化，虽然有时会带来一定正则化效果。【应用场景】CNN 和深层前馈网络中常使用 BatchNorm 来改善训练稳定性。
keywords: BatchNorm, 批归一化, Mini-batch, 训练稳定性, 深度学习
```

## 6.7 CNN

```text
knowledge_node_id: ml_cnn
source_ids: book_d2l, book_nndl_qiu, course_stanford_cs231n
title: CNN 的局部连接与参数共享
importance: 0.80
content: 【核心概念】卷积神经网络通过卷积层在局部区域提取特征，并让同一个卷积核在不同位置共享参数。相比全连接网络，CNN 更适合图像等网格结构数据，因为它利用了局部相关性和平移模式。【直觉理解】识别图片时，边缘、纹理、角点等局部模式可能出现在任意位置；卷积核像一个小探测器，在整张图上滑动寻找相似模式。【与前置知识的关系】需要先理解多层神经网络和反向传播；卷积层、池化层和全连接层共同组成典型 CNN。【典型误区】卷积不是简单压缩图片，池化也不是必须越多越好；CNN 的优势来自结构先验而不是只因为参数更多。【应用场景】图像分类、目标检测、医学图像分析和视觉特征提取都广泛使用 CNN。
keywords: CNN, 卷积, 池化, 局部连接, 参数共享, 图像分类
```

## 6.8 RNN

```text
knowledge_node_id: ml_rnn
source_ids: book_d2l, book_nndl_qiu
title: RNN 如何处理序列数据
importance: 0.70
content: 【核心概念】循环神经网络用于处理序列数据，它在每个时间步接收当前输入，并结合上一个时间步的隐藏状态产生新的隐藏状态。隐藏状态像对历史信息的压缩记忆，使模型能够根据上下文进行预测。【直觉理解】阅读一句话时，理解当前词往往依赖前面的词；RNN 就像一边读一边更新笔记，把过去信息带到后续判断中。【与前置知识的关系】需要先理解多层神经网络、反向传播和梯度传递；RNN 的训练通常使用沿时间展开后的反向传播。【典型误区】RNN 并不能自然记住任意长的历史，长序列中容易出现梯度消失或爆炸；隐藏状态也不是完整保存所有过去输入。【应用场景】语言建模、时间序列预测、语音识别和早期序列标注任务常使用 RNN 或其变体。
keywords: RNN, 序列建模, 隐藏状态, 时间步, 梯度消失
```

## 6.9 LSTM

```text
knowledge_node_id: ml_lstm
source_ids: book_d2l, book_nndl_qiu
title: LSTM 的门控记忆机制
importance: 0.65
content: 【核心概念】LSTM 是 RNN 的重要变体，通过输入门、遗忘门、输出门和记忆单元控制信息写入、保留与输出。它的设计目标是缓解普通 RNN 在长序列中难以保留长期依赖的问题。【直觉理解】普通 RNN 像每读一个新词就覆盖笔记，LSTM 则多了几个开关：哪些旧信息继续保存，哪些新信息写入，哪些记忆用于当前输出。【与前置知识的关系】需要先理解 RNN、隐藏状态和梯度消失问题；LSTM 是进入注意力机制和更现代序列模型前的重要历史节点。【典型误区】LSTM 不是完全解决所有长程依赖问题，也不一定在所有任务上优于更简单模型；门控机制增加了参数和计算成本。【应用场景】文本序列、语音、时间序列和需要较长上下文记忆的任务曾广泛使用 LSTM。
keywords: LSTM, 门控机制, 长程依赖, 记忆单元, 序列模型
```

## 6.10 注意力机制

```text
knowledge_node_id: ml_attention_mechanism
source_ids: book_d2l, book_nndl_qiu, course_hung_yi_lee_ml
title: 注意力机制的加权选择思想
importance: 0.85
content: 【核心概念】注意力机制通过为不同输入位置分配权重，让模型在生成当前表示时重点利用更相关的信息。常见形式中，Query 表示当前要查询什么，Key 表示各位置可被匹配的索引，Value 表示实际被汇总的信息。【直觉理解】回答问题时，人不会平均阅读所有文字，而会把注意力放在最相关的句子上；注意力机制就是让模型学习这种“按相关性加权读取”的过程。【与前置知识的关系】需要先理解向量、矩阵乘法和序列模型；它是 Transformer 自注意力的直接前置。【典型误区】注意力权重不等于人类可解释性的全部；注意力本身也不天然包含顺序信息，序列位置通常还需要额外编码。【应用场景】机器翻译、文本生成、图像描述、语音和多模态任务中都广泛使用注意力机制。
keywords: 注意力机制, Query, Key, Value, 权重, 加权求和
```

## 6.11 Transformer

```text
knowledge_node_id: ml_transformer
source_ids: book_d2l, book_nndl_qiu, course_mit_6s191
title: Transformer 的自注意力架构
importance: 0.80
content: 【核心概念】Transformer 是基于自注意力机制的深度学习架构，核心模块包括多头自注意力、前馈网络、残差连接、归一化和位置编码。它不依赖 RNN 的逐步递归结构，而是让序列中不同位置可以直接通过注意力建立联系。【直觉理解】RNN 像按顺序传话，远处信息可能在传递中变弱；Transformer 更像让所有位置开会，每个位置都能根据相关性直接听取其他位置的信息。【与前置知识的关系】需要先理解多层神经网络、注意力机制和位置表示；它是现代大语言模型的重要基础。【典型误区】Transformer 不是不需要训练数据，也不一定在任何小任务上都优于简单模型；自注意力还需要位置编码或类似机制来表达顺序。【应用场景】机器翻译、文本生成、问答系统、代码模型、视觉 Transformer 和多模态大模型都以 Transformer 为核心或重要组件。
keywords: Transformer, 自注意力, 多头注意力, 位置编码, 大模型
```
