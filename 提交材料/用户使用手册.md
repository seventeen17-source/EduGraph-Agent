# EduGraph-Agent 用户使用手册

版本：1.0 · 日期：2026-07-16

---

## 1. 系统简介

EduGraph-Agent 是一个面向《机器学习》课程的个性化学习平台。你可以：

- 和 **学习助手** 对话，获得基于知识图谱证据的可追溯讲解；
- 一键生成 **6 类学习资源**（文档 / 思维导图 / 练习题 / 视频脚本 / 代码案例 / 教学图片）；
- 在 **知识图谱** 上直观查看知识点依赖关系与自己的学习进度；
- 通过 **练习测试** 巩固知识，成绩自动回写学习画像；
- 获得基于诊断的 **个性化学习路径** 推荐；
- 在 **学习成长** 页回顾学习轨迹、接收遗忘预警。

## 2. 环境准备与启动

> 已部署好的环境可跳过本节，直接从第 3 节开始。

### 2.1 前置条件

- Python 3.11+、Node.js 18+
- Neo4j 5.x 已启动（默认 `bolt://localhost:7687`，账号 `neo4j / edugraph123`）
- 有效的 LLM API Key（OpenAI 或 DeepSeek 兼容端点）

### 2.2 首次部署

```bash
# 1) 导入知识图谱数据
python Scripts/import_to_neo4j.py --uri bolt://localhost:7687 --user neo4j --password edugraph123 --drop-existing

# 2) 配置环境变量：复制模板并填入 API Key
#    模板：提交材料/部署配置/.env.example → 复制为 backend/.env

# 3) 启动后端（终端 1）
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000

# 4) 启动前端（终端 2）
cd frontend
npm install
npm run dev
```

验证：浏览器打开 `http://127.0.0.1:8000/health` 应返回 `{"status":"ok","neo4j":"ok"}`；然后访问前端 `http://127.0.0.1:5173`。

Docker 方式部署见《部署配置/部署说明.md》。

## 3. 快速上手

### 3.1 登录 / 注册

- **演示账号**：登录页点击"以演示账号体验"（邮箱 `demo@edugraph.local`，密码 `demo123`）；
- **注册新用户**：点击"去注册"，填写用户名 / 邮箱 / 密码，注册成功后自动跳转画像对话页。

登录态自动保持，刷新页面不丢失。

### 3.2 建立学习画像（首次使用）

注册后进入 **画像对话** 页：

1. 用自然语言描述你的学习背景，例如："我是大三学生，学过高数和线代，Python 基础还行，想在两个月内系统学完机器学习，目标是做课程设计。"
2. 系统自动抽取 8 维画像（目标、基础、风格、时间投入等），页面顶部 **画像完整度** 上升；
3. 可多轮补充；也可在 **画像面板** 页手动编辑任何字段。

画像越完整，后续讲解、资源和路径推荐越贴合你。

## 4. 功能使用指南

### 4.1 学习助手（核心）

进入 **学习助手** 页：

1. 输入问题，如"帮我讲解反向传播"；
2. 右侧 **AgentTracePanel** 实时显示 17 个智能体节点的执行过程（意图识别 → 证据检索 → 质量评估 → 回复合成…）；
3. 回复以打字机效果流式输出，支持 Markdown、数学公式、代码高亮；
4. **EvidencePanel（紫色面板）** 展示本次回答依据的知识图谱多跳依赖路径——你可以看到"学反向传播需要先懂链式法则和梯度下降"这类前置关系；
5. 对每条回复可点 **反馈标签**（有帮助 / 没看懂 / 太深了 / 太浅了 / 例子不够…共 8 种），系统据此更新你的行为画像，下次回答自动适应你的偏好。

提示：宽泛问题（如"机器学习的应用"）会得到主题级回答和更具体的备选问题建议。

### 4.2 知识图谱

进入 **知识图谱** 页：

- **搜索**：按名称/别名/关键词模糊搜索知识点；
- **筛选**：按章节、难度滑块过滤；
- **节点着色**：颜色代表你的掌握状态（未学 / 学习中 / 已掌握 / 遗忘预警）；
- **点击节点**：查看详情、前置依赖路径、关联练习与代码案例；
- 顶部仪表盘显示总体学习进度与个性化推荐起点。

### 4.3 AI 资源生成

进入 **资源生成** 页：

1. 搜索并选择知识点（如"K-Means 聚类"）；
2. 勾选需要的资源类型，点击生成；
3. 6 类资源并发生成，进度面板实时显示各智能体状态：
   - **讲解文档**：结构化图文讲解；
   - **思维导图**：可交互展开/折叠（markmap）；
   - **练习题**：可直接作答；
   - **视频脚本**：分镜脚本，可用于录制微课；
   - **代码案例**：可运行的 Python 示例，语法高亮；
   - **教学图片**：AI 文生图（讯飞 HiDream）；
4. 所有生成的资源自动保存到 **知识中心**，可随时回看，思维导图支持二次编辑。

### 4.4 练习测试

进入 **练习** 页：

1. 搜索选择知识点，选择练习模式（4 种：快速练习 / 章节测验 / 错题重做 / 复习推荐）；
2. 逐题作答：选择题即时判分；简答 / 代码 / 案例分析题由后端 LLM 评分并给出详细反馈；
3. 整轮提交后查看成绩汇总；
4. 成绩自动回写画像的知识点掌握度（带证据链），错题进入 **错题本**；
5. **练习历史** 页可回看全部练习记录。

### 4.5 评估报告

进入 **评估** 页：掌握度 **雷达图**、章节 **柱状图**、题型 **饼图**，直观定位薄弱点。

### 4.6 学习路径

进入 **学习路径** 页：

- 系统基于诊断结果（薄弱点、遗忘点、目标依赖）用拓扑排序生成推荐队列；
- 每个节点附 **推荐理由**（如"该节点是你目标『神经网络』的前置，且上次练习正确率 40%"）；
- 5 种推荐类型标注：补弱 / 复习 / 目标推进 / 新知 / 遗忘唤醒。

### 4.7 学习成长

进入 **学习成长** 页：

- **日历热力图**：每天学习强度一目了然；
- **事件时间轴**：每次对话、练习、生成资源、画像更新均可追溯；
- **遗忘预警**：基于遗忘曲线提示该复习的知识点；
- **周报**：每周学习摘要。

### 4.8 管理面板（教师/管理员）

进入 **Admin** 页：查看全体反馈统计、LLM 批量反馈分析结果、系统运行状态（Neo4j / SQLite / LLM / Embedding / ChromaDB 健康度）。

## 5. 常见问题（FAQ）

| 问题 | 解决方法 |
|------|---------|
| 前端报 Network Error | 检查后端是否启动：访问 `http://127.0.0.1:8000/health` |
| 搜索知识点无结果 | 确认 Neo4j 已启动且已运行导入脚本；搜索支持名称/别名/关键词/UID 模糊匹配 |
| 回复提示 LLM 不可用 | 检查 `backend/.env` 中 `LLM_API_KEY` 是否有效、余额是否充足 |
| 图片生成失败 | 检查讯飞 HiDream 配置；其余 5 类资源不受影响 |
| Windows 下 npm 报执行策略错误 | 使用 `npm.cmd` 代替 `npm` |
| 登录后刷新被登出 | 清除浏览器缓存后重新登录（token 会自动刷新，正常不应发生） |

## 6. 推荐体验路线（约 7 分钟）

```
登录（演示账号） → 画像对话（描述背景，看完整度上升）
  → 学习助手（问"帮我讲解反向传播"，看 17 节点执行 + 多跳依赖面板）
  → 知识图谱（搜索节点，看进度着色与前置路径）
  → 资源生成（生成 6 类资源）
  → 练习测试（做几题，看即时评分）
  → 学习路径（看拓扑排序推荐与理由）
  → 反馈标签（点"没看懂"，看行为画像变化）
  → 学习成长（热力图 + 遗忘预警） → 评估报告（雷达图）
```
