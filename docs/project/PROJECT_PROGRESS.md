## 3. 核心闭环

### 3.1 学习闭环
```
注册/登录 → 画像建立(8维) → LangGraph对话
  → GraphRAG多跳证据检索 → 6类AI资源生成 → 练习评估(4模式)
  → 诊断推荐(Kahn拓扑排序) → 学习路径(推荐队列) → 画像更新(证据链)
```

### 3.2 反馈闭环
```
学生点反馈标签 → POST /feedback → BehaviorProfileUpdater(EMA更新)
  → 下次对话 load_context → behavior注入prompt → 适应摘要展示
```

### 3.3 记忆闭环
```
对话完成 → extract_memory → LLM提取MemoryEntry → ChromaDB insert
  → 下次对话 retrieve_memory → 混合检索 → 注入prompt
```
