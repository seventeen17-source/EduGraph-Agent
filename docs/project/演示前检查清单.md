## 四、演示流程（7分钟）

1. [ ] **登录** → 演示账号一键登录 或 注册新用户
2. [ ] **画像对话**（30s）→ 自然语言描述学习背景，画像完整度上升
3. [ ] **学习助手对话**（1min）→ 提问"帮我讲解反向传播"
   - 观察 AgentTracePanel 18节点实时执行
   - 查看 EvidencePanel 多跳依赖分析
   - 查看回复的打字机效果
4. [ ] **知识图谱**（30s）→ 搜索节点/子图展开/前置路径
5. [ ] **资源生成**（1min）→ 生成5类学习资源
6. [ ] **练习测试**（1min）→ 做题 + 即时反馈 + 画像回写
7. [ ] **学习路径**（30s）→ 查看拓扑排序推荐
8. [ ] **反馈闭环**（30s）→ 给回复打标签"没看懂"/"有帮助"
9. [ ] **学习成长**（30s）→ 日历热力图 + 遗忘预警
10. [ ] **评估报告**（30s）→ 掌握度雷达图 + 薄弱点
